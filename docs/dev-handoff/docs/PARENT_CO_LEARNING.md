# Parent Co-Learning Mode

## Philosophy

Parents should not become pressure machines or substitute tutors. The app should help parents become mentors who ask better questions, notice patterns, and protect the child's wellbeing.

## Daily parent mission

Each day, parent receives one 5-15 minute mission:

- Listen to child explain one idea.
- Read one page together.
- Ask one open question.
- Review one mistake without blame.
- Do one small project task.
- Walk and discuss one concept.

## Weekly family review

Agenda:

1. What did you learn this week?
2. What mistake helped you grow?
3. What was hard?
4. What did you enjoy?
5. What should we do next week?
6. What do you need from mom/dad?

## Parent dashboard design

Show:

- One highlight.
- One concern.
- One suggested action.
- Mastery trend.
- Mistake patterns.
- Reading trend.
- Wellbeing signal.

Do not show:

- Public rankings.
- Excessive analytics.
- Shame-based warnings.

## Parent prompt examples

Instead of: "Why did you get this wrong?"
Use: "Which step confused you?"

Instead of: "Let me do it for you."
Use: "What have you tried? What is the next tiny step?"

Instead of: "You are careless."
Use: "This looks like a reading-the-question error. How can we catch it next time?"

## Media plan

Build a family media plan with:

- No device during meals.
- No device before sleep window.
- Learning mode separate from entertainment mode.
- Parent approval for AI use under 13.
- Weekly screen-time review.
