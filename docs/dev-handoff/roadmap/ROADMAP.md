# Roadmap

## Phase 0 - Prototype preparation

Deliverables:
- Repo setup.
- Design system.
- Data schema.
- AI provider adapter.
- Safety middleware skeleton.
- Seed content for Grade 1-2 Math, Vietnamese reading, English basics.

## Phase 1 - Private MVP for one child

Timeline target: build enough for daily use.

Features:
1. Parent onboarding.
2. Child profile.
3. Daily plan.
4. Lesson session UI.
5. AI Socratic tutor.
6. Hint ladder.
7. Mistake notebook.
8. Spaced review.
9. Reading journal.
10. Parent dashboard.
11. Weekly review.
12. Safety audit metadata.

Acceptance:
- Child can complete a 20-minute session.
- Parent can review progress in under 3 minutes.
- Mistakes generate review tasks.
- AI does not give direct answers first.

## Build strategy

Do not build every subject first. Build the learning loop first:

Plan -> Attempt -> Hint -> Mistake -> Review -> Mastery -> Parent insight -> Reflection.

## Cross-cutting roadmap update: Adaptive acceleration

Adaptive acceleration is not a later luxury. It should be included as a core design principle from MVP because the product goal is to serve one real child, not an average child.

### MVP inclusion

- Store domain-specific skill states.
- Do not hard-code grade ceilings.
- Track hints, retention, transfer, and explanations.
- Generate basic compact/enrich/scaffold recommendations.

### V2 inclusion

- Full asynchronous profile dashboard.
- Challenge Fit Score visualization.
- Advanced enrichment/project generator.

### V3 inclusion

- Longitudinal talent development map.
- Domain-specific acceleration planning.
- Parent decision support for school/pathway choices.

## Phase 2.5 - Owner capability track

Add the six-capability track after core mastery, parent co-learning, and safety are stable.

Scope:

- Owner Mindset Map.
- Decision Lab.
- Capital Allocation Simulator.
- Pitch & Negotiation Studio.
- Rule-Maker Lab.
- Ethics Under Pressure.

Do not add this as a status/wealth game. It must be framed as agency, responsibility, value creation, and stewardship.
