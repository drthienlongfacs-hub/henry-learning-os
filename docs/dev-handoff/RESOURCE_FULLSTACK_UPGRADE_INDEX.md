# Resource Full-Stack Upgrade Index

This package now includes a full resource and integration layer for every major module.

New files:

- `resources/RESOURCE_BENCHMARK_FULLSTACK.md` - module-by-module benchmark, APIs, GitHub repos, OER, data sources, and feature translation.
- `resources/INTEGRATION_CATALOG.json` - machine-readable catalog for coding agents.
- `resources/API_EVALUATION_MATRIX.md` - practical integration priority/risk matrix.
- `architecture/RESOURCE_INTEGRATION_ARCHITECTURE.md` - adapter architecture, schemas, safety gates, cache rules.
- `issues/RESOURCE_INTEGRATION_BACKLOG.md` - development issues for integrating resources.
- `prompts/RESOURCE_HARVEST_AND_INTEGRATION_MISSION.md` - prompt for Antigravity/Codex to use the resource layer.

Rule: external resources are raw material, not the product. The app's advantage comes from wrapping resources in child-safe pedagogy, adaptive challenge-fit, data-driven mastery, parent co-learning, and long-term portfolio evidence.
